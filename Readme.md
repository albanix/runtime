Я не впевнений, що буде робити цей проєкт для масс, просто для себе, мабуть навіть крейта на карго не буде, мабуть, хоча якщо буду бачити сенс.
тоож що ми маємо?
на момент версії 1.0.0:

```rs
fn main() {
    let rt = Arc::new(Executor {
        queue: Mutex::new(VecDeque::new())
    }); // організовуємо рантайм, обов'язово Arc.

    rt.spawn(async {
        println!("Hello from task 1");
    }); // кладемо першу таску в стек

    rt.spawn(async {
        println!("Hello from task 2");
    }); // кладемо другу таску у стек
    rt.run();

    // особливої різниці не буде, потрібно використати щось дійсно важке, а не прінт в консоль
}```
